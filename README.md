# Introduction

In this example role, you will learn how to write a role, test cases and run them.

# Requirements

* Vagrant
* Virtualbox (>= 5.0.x, 4.x might work but the virtual box plugin on the VM box is version 5.0.x)
* Ansible (>= 2.0.x)
* bundler
* git
